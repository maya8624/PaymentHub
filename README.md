This project is a simple and secure payment system for handling online transactions. 
It lets you create orders, approve payments, capture funds, and issue refunds. 
Currently, it integrates with PayPal Sandbox for testing purposes.

1. Features
- Create and manage orders
- Approve and capture payments
- Refund transactions
- Frontend: Planned / Not Yet Implemented
- Webhook support for real-time updates: Planned / Not Yet Implemented
- Authentication: Planned / Not Yet Implemented
- Azure deployment: Planned / Not Yet Implemented

2. Technologies
- C# .NET 8 / ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- PayPal Sandbox API
- FluentValidation
- Swagger

⚠️ Note: This project is still a work in progress. Features are being expanded, and improvements are ongoing to support more payment providers and robust error handling and testing.
